# Guide SEO - Out of the Books

## Configuration globale

Paramètres dans `src/content/site/settings.json`, modifiables via Tina CMS ("Paramètres généraux").

- **Template de titre** : `%s | Out of the Books`
- **Description par défaut** : "Out of the Books ASBL : l'éducation et le bien-être de l'enfant..."
- **Langue** : `fr`
- **Image sociale par défaut** : configurable dans Tina

## Métadonnées par page

Chaque page peut surcharger les paramètres globaux. Fichiers : `src/content/[page-name]/index.json`

### Champs personnalisables

- **Titre SEO** : 50-60 caractères, sans "| Out of the Books" (ajouté automatiquement)
- **Description SEO** : 150-160 caractères
- **Image de partage** : 1200x630px recommandé

### Valeurs par défaut

| Page | Titre | Description |
| ---- | ----- | ----------- |
| À propos | "À propos" | Équipe et mission ASBL |
| Contact | "Contact" | Informations de contact |
| Blog | "Nos contenus" | Podcasts, vidéos, fiches |
| Appel à projets | "Appel à projets" | Soumission projets pédagogiques |
| Festival | "Festival Out of the Books" | Innovation pédagogique |

## Modification via Tina CMS

1. Se connecter à Tina CMS
2. Naviguer vers la page cible
3. Section "Métadonnées SEO"
4. Laisser vide = valeur globale par défaut

## Bonnes pratiques

- Titres : mots-clés pertinents, naturels et accrocheurs
- Descriptions : résumé clair + appel à l'action
- Images : qualité 1200x630px pour les partages sociaux
- Cohérence entre titre/description et contenu réel
